Compile Sass to Css

Compile sass to css with this command: 

      sass --watch style.scss:style.css --no-cache --style compressed

Nesting

Attention! Avoid nesting below 3 levels. 2-3 levels is sufficient to achieve your golas.

CSS

      h1 { 
        font-size: 3rem;
        color: red; 
      }
      h1 a {
        color:blue;
        }
      h1 a:hover {
        color: green;
      }

Sass

      h1 {
        font-size: 3rem;
        color: red;
        a {
            color:blue;
            &:hover {
              color: green;
            }
          }

@extend

Define some class eg. btn and then use it in other classes by adding @extend .btn

For example if you have a button with a class .btn

And then want to have a div with class of .colorful 

    .btn {
      background-color: $accent;
      color: $basecolor;
      font: 2em $basefont 900;
      border-radius; $radius;
      border: 0;
      padding: $basesize $basesize * 2;
    }

    .colorfull {
      @extend .btn;
    }

Transitions

Transitions are really simple o make. For example if you want to change the background color of a button on hover. 


      div {
        transition: all 0.5s ease;
        background: tomato;
        padding: 10px;
          &:hover {
          background: lightgray;
          color: black
          padding: 20px;
          }
        }

@extend and %placeholder

Define your "abstract class (not present in HTML code called eg.transition.

Use that clas later to @extend from it

    %transition
       {
        background-color: $success;
        border-radius: 0.5em;
        box-shadow: 2px 2px 6px -2px rgba(0, 0, 0, 0.3);
        color: $pc;
        transform: scale(1.02);
      }
      .butt-info {
        transition: all 1s ease-in-out;
        background-color: $info;
        border-radius: 0.5em;
        &:hover {
          @extend %transition;
          background-color: darken($info, 20%);
          cursor: pointer;
        }
      }

Parent selectors

h2 {
  color: white;
  .about & {color: red;}
  .contact & {color:orange;}
  .info & {color:yellow;}
}

Compiles to:

h2 (color:white;)
  .about h2 {color: red;}
  .contact h2 {color: orange;}
  .info h2 {color: yellow;}
h2 {
  color: white;
  .about {color: red;}
  .contact {color: orange;}
  .info {color: yellow;}
}

Compiles to:

Anything that is inside h2 with classes of .about, .contact, .info as a child element

  h2 (color:white;)
  h2 .about {color: red;}
  h2 .contact {color: orange;}
  h2 .info {color: yellow;}

  h2 {
  color: white;
  &.about {color: red;}
  &.contact {color: orange;}
  &.info {color: yellow;}
}

Compiles to:

Anything that is at the samee level as h2 with classes of .about, .contact, .info as a child element

h2 (color:white;)
h2.about {color: red;}
h2.contact {color: orange;}
h2.info {color: yellow;}
a {
  color: white;
  &:hover {
  color:red;
  }
}

Compiles to:

a {color:white;}
a:hover {color:red;}

@import to work with multiple files

Create a partial file with a name of _file.scss

and then import it into the main sass file:

@import "file";

    @import "filename";
